# gauge-canvas

This is a library drawing different type of gauges using canvas.

## Install

Install via `bower`:

```shell
bower install gauge-canvas
```

Add a `<script>` to your `index.html`:

```html
<script src="/bower_components/gauge-canvas/gauge-canvas.js"></script>
```
